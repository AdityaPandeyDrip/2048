# 2048
2048 game made in java , run the Main.java file to start the game.  
Changes can be made in the size of the board and winning value of the cells.

## Game Instructions
| Move    | Key to Press  | 
| :---:   | :-: | 
| To Quit | 0 |
| Left Shift | 1 |
| Right Shift | 2 |
| Up Shift | 3 |
| Down Shift | 4 |
